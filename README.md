
# FILend | A Lending Dapp



## Procedure 

* Navigate to your cloned repository  and install the `npm tool`:

```
npm install
```

Dependecies packages will be downloaded.

* Start DApp on localhost server:

```
start run dev
```


* For a production build:

```
npm dev  build
```



## Functionality
- Lending order 
- Borrow request
- Loan payback 
- Withdraw
